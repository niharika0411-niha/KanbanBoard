This project is an Interactive Kanban Board built using ReactJS, based on the assignment requirements from the provided API. The application allows users to display and organize tasks in a Kanban board format, dynamically adjusting the display based on various grouping and sorting options.
This project demonstrates the power of ReactJS in building dynamic, user-friendly frontend applications. The Interactive Kanban Board is an effective way to manage tasks, visualize project progress, and enhance productivity.
